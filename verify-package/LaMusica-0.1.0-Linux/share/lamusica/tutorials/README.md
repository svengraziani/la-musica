# Tutorial Fixtures

Tutorial fixtures must load without developer tools and must not depend on non-redistributable
media. Current tutorial projects use generated or empty clip references only, so no external audio
files, plugin binaries, or sample packs are shipped.

Current tutorials:

- `fixtures/tutorials/first-song.Project.lamusica`

`first-song.Project.lamusica` provides a small arrangement with markers, audio and MIDI tracks,
clips, and routing for first-run documentation and packaging smoke checks.
