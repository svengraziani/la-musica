# Code Of Conduct

LaMusica follows the Contributor Covenant Code of Conduct.

## Standard

Contributors are expected to collaborate respectfully, focus on technical substance, and keep discussions constructive and relevant to the project.

Unacceptable behavior includes harassment, discriminatory language, personal attacks, deliberate disruption, and publishing private information without consent.

## Reporting

Report conduct issues privately to the project maintainers once maintainer contact details are published.

Until maintainers publish dedicated contact information, do not include private reports in public issues.
