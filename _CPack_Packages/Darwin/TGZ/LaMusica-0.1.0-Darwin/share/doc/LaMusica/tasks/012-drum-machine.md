# 012 Drum Machine

## Objective

Create an integrated drum machine for sample-based drum programming.

## Dependencies

- 010 MIDI Core.

## Deliverables

- Drum rack with pads, sample assignment, choke groups, mute/solo, pad colors, velocity layers, and per-pad output routing.
- Built-in sample playback engine with envelopes, pitch, filters, reverse, and start/end controls.
- Pad browser and drag/drop from project assets.
- Preset format and bundled open-license starter kit placeholders.
- Tests for sample triggering, choking, routing, and preset load/save.

## Acceptance Gates

- Pads trigger sample playback with predictable latency.
- Choke groups and per-pad routing work in live and offline render.
- Presets remain portable across machines when assets are collected.

## Notes For Agents

Bundled sounds must have clear redistribution rights.
